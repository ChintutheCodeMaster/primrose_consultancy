# Primrose IEC

**The CRM for independent educational consultants.**

Manage inquiries, students, college lists, applications, essays, engagement agreements and billing in one calm, organized workspace — purpose-built for the way independent educational consultants actually work.

## Routes

- `/` — public marketing site
- `/app` — authenticated dashboard (password-gated until multi-tenant auth ships)

## Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Lovable Cloud (Supabase) for database, storage, edge functions and AI

## Status

This project is mid-pivot from the original Hebrew "Noga" study-abroad CRM into the English multi-tenant Primrose IEC product. See `.lovable/plan.md` for the 5-phase roadmap.
